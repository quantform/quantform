## `getInstrument`

| Parameter | Type | Description |
|-----------|------|-------------|
| instrument | InstrumentSelector | An object containing the instrument identifier to search for |

Emits a single Instrument instance matching the provided selector.

```typescript
function getMakerFee() {
    const { getInstrument } = useBinance();
    
    // get maker trading fee for specific market
    return getInstrument(instrumentOf('binance:eth-usdt')).pipe(
      map(it => it.commission.makerRate)
    );
}
```

## `getInstruments`

Emits an array of all available trading instruments on market:

```typescript
function getAllInstruments() {
    const { getInstruments } = useBinance();

    // get all available usdt quote markets
    return getInstruments().pipe(
      map(it => Object.values(it).filter(it => it.quote.id == 'usdt'))
    );
}
```
