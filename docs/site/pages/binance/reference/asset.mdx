## `getAsset`

| Parameter | Type | Description |
|-----------|------|-------------|
| asset | AssetSelector | An object containing the asset ID to retrieve |

Emits a single Asset instance matching the provided asset selector.

```typescript
function getAssetNativeId() {
    const { getAsset } = useBinance();

    return getAsset(assetOf('binance:btc')).pipe(map(it => it.id));
}
```

## `getAssets`

| Parameter | Type | Description |
|-----------|------|-------------|
| - | - | No parameters required |

Emits a collection of all Asset available on spot market.

```typescript
function listAllAssets() {
    const { getAsset } = useBinance();

    return getAssets().pipe(map(it => Object.keys(it)));
}
```
